# Product Overview

## Yoga Booking System

A comprehensive yoga studio management and booking platform built for modern yoga studios. The system handles class scheduling, student bookings, payment processing, and administrative management.

### Core Features

- **Class Management**: Schedule yoga classes with different types, instructors, and capacities
- **Booking System**: Students can book classes, join waitlists, and manage their reservations
- **Payment Integration**: Stripe-powered payment system supporting multiple pass types (drop-in, 5-class packs, unlimited monthly)
- **User Authentication**: Secure login with credentials and Google OAuth
- **Administrative Dashboard**: Studio management tools for sessions, instructors, and bookings
- **Waitlist Management**: Automatic waitlist handling when classes reach capacity

### Pass Types

- **Drop-in**: Single class payment
- **Pack of 5**: Credit-based system for 5 classes
- **Unlimited Monthly**: All-access monthly subscription

### Target Users

- **Students**: Book and manage yoga class reservations
- **Instructors**: View their scheduled classes and student lists
- **Studio Administrators**: Manage all aspects of studio operations

The system is designed to be scalable, user-friendly, and integrate seamlessly with existing studio workflows.
