## Usage Guidelines

### Colors
- Use **terracotta** for primary actions and brand elements
- Use **sage** for secondary actions and accents
- Use **sand** for text and neutral elements
- Use **cream** for backgrounds and surfaces
- Use **clay** for warm accents and highlights

### Typography
- Use **Playfair Display** for headings and display text
- Use **Inter** for body text, labels, and UI elements
- Maintain proper contrast ratios (4.5:1 for normal text, 3:1 for large text)

### Spacing
- Use consistent spacing values from the scale
- Prefer larger spacing values for better breathing room
- Use smaller spacing for compact layouts when necessary

### Shadows
- Use subtle shadows to create depth
- Avoid heavy shadows that conflict with the natural aesthetic
- Use stronger shadows sparingly for important overlays
